Troubleshooting – Line 2 Metrology & Inspection

Last updated: Oct 5, 2025 — Priya Shah • Watchers: 18
Scope: CD-SEM-05, AOI-11, Ellips-02, Prof-03. Use the decision flow to triage, then expand the relevant symptom below.

Decision Flow

1) Is there a safety concern?

HF/NH₄OH, fumes, spill, or LOTO tags → STOP and contact EHS.

2) Is the lot on hold?

Check QMS tags. If yes, escalate before measurement.

3) Tool status green?

Verify PM/cals & interlocks. If not, call Equip Eng.

4) Validate reference

Run REF wafer first; compare to latest baseline.

Common Symptoms (expand to view steps)

CD-SEM results drifting high (+0.5–1.0 nm)
Likely CausesStigmation/focus off; dirty aperture; recipe miss match; stage drift.
Quick ChecksRun REF wafer (STD-REF-02 center + 4 corners). Compare μ/σ vs last baseline. Verify magnification and WD.
FixAuto-stigmation → Re-focus → Clean aperture → Reload recipe v3.2 → Re-run REF. If Δ>1.0 nm persists, escalate.
# CD-SEM quick baseline (pseudo)
load_wafer STD-REF-02
set_recipe M8_TOP_v3.2
measure center, q1, q2, q3, q4
report --stats --export \\QA-SHARE\Metrology\CDSEM\baseline.csv
AOI shows particle spike (edge-ring)
Likely CausesCMP pad wear; post-etch residue; foreline particles.
Quick ChecksCompare last good vs current wafer maps; inspect pad life; check RIE foreline PM schedule.
FixReplace CMP pad → Rinse audit → AOI re-scan. If density >0.20/cm² after PM, start 8D.
# AOI delta vs baseline
aoi compare --lot W78A-024 --pre last_good --post current --zones edge,center
aoi export --wafer-map AOI_W78A_Overview.png
Ellipsometer thickness drift (+40–100 Å)
Likely CausesWrong model; reference aged; angle mis-set.
Quick ChecksRun ELL-REF-A twice; Δ between runs ≤ 20 Å; verify angle 70° (or recipe default).
FixUpdate offset per SOP 4.6; replace/clean reference wafer; document ECN note.
# Ellipsometer two-pass verification
ell set recipe THK_5200A
ell measure REF-A --repeat 2
ell calc --offset --apply
ell save \\QA-SHARE\Metrology\Ellips\2025_10_05.csv
Profilometer step height OOT (+0.06–0.10 µm)
Likely CausesTip wear; stage tilt; wrong force setpoint.
Quick ChecksGauge blocks GB-1.00 / GB-1.10; stage planarity test; verify force.
FixReplace stylus → Planarity routine → Re-cal. Keep lots on hold until within ±0.05 µm.
# Profilometer calibration
prof lockout on
prof planarity --auto
prof calib --blocks GB-1.00,GB-1.10
prof verify --block GB-1.05
prof lockout off
Litho overlay p95 > 0.035 µm (LTH-09)
Likely CausesCompensation table stale; reticle pins; focus uniformity.
Quick ChecksOVX/OVY trend; reticle seating; stage temp; recipe version.
FixDeploy v3.2-h1 compensation; verify on 3-wafer DOE; escalate if p95 persists > 0.035 µm.
# Litho compensation push + verify
lth set-recipe M8_TOP_v3.2-h1
lth run-doe --wafers 3 --metrics OVX,OVY
lth export --spc \\QA-SHARE\Photo\LTH09\overlay_trend.csv
