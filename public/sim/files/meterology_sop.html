Standard Operating Procedure (SOP)

Document ID: MET-SOP-102
Rev: 3.1
Code:

1. Purpose

To establish the standardized procedure for operating and maintaining metrology equipment (CD-SEM, Ellipsometer, and Profilometer) used for line width, film thickness, and step height measurements in production wafers. This SOP ensures data accuracy, repeatability, and compliance with QA specifications.

2. Scope

Applies to all QA and Process Engineering staff performing inline and post-process metrology at FAB Line 2, 200 mm wafers. This includes CD-SEM-05, Ellips-02, and Prof-03 tools.

3. Responsibilities

RoleResponsibilities
QA TechnicianExecute measurement per defined sampling plan, record data in QMS, and flag out-of-control (OOC) results.
Process EngineerReview SPC trends, verify calibration results, and approve recipe changes.
Equipment EngineerPerform periodic maintenance, calibration, and software verification.
QA SupervisorEnsure compliance and approve any deviations or exception reports.

4. Procedure Overview

StepDescriptionFrequency
4.1Verify tool status and confirm calibration sticker is valid.Before each use
4.2Load sample wafer using vacuum wand; ensure chuck cleanliness.Each run
4.3Run CD measurement on 5 die locations (center + 4 corners).Each wafer
4.4Log data into QMS system under lot ID and operator ID.Each run
4.5Perform gauge repeatability and reproducibility (GR&R) check weekly.Weekly
4.6Initiate tool calibration using reference wafer (STD-REF-02).Monthly

5. Acceptance Criteria

ParameterNominalToleranceAction if Out of Spec
CD (nm)78±1.0Stop tool; notify Process Eng. for recipe check.
Film Thickness (Å)5200±100Run 2nd measurement, confirm reference wafer.
Step Height (µm)1.05±0.05Flag lot; escalate to QA Supervisor.

6. Safety & Precautions

  • Ensure all wafers are at room temperature before loading.
  • Use antistatic gloves and wafer tweezers to prevent contamination.
  • Never touch the tool lens or chuck surface directly.
  • Report any vacuum or stage anomalies immediately to Equipment Engineering.
